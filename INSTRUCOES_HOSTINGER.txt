INSTRUÇÕES PARA HOSPEDAGEM NO HOSTINGER:

OPÇÃO A: USANDO O ARQUIVO ZIP (Mais fácil e rápido)
1. Baixe o arquivo .ZIP do projeto aqui no editor.
2. Extraia o ZIP e entre na pasta 'dist'.
3. No painel da Hostinger, abra o 'Gerenciador de Arquivos' e entre na 'public_html'.
4. Envie todo o CONTEÚDO da pasta 'dist' para dentro da 'public_html'.
   - O arquivo 'index.html' deve ficar direto na pasta principal.

OPÇÃO B: VIA GITHUB
1. Crie um repositório no GitHub e suba todos os arquivos do projeto.
2. IMPORTANTE: Para hospedagem comum (Shared), você deve rodar o comando 'npm run build' localmente e garantir que a pasta 'dist' seja enviada ou use uma 'GitHub Action' para automatizar o envio para o FTP da Hostinger.
3. O site já está configurado com '.htaccess' e 'base path' corretos para aceitar seu domínio próprio.

DICA: Se os links não funcionarem ou derem erro 404 ao atualizar, verifique se o arquivo '.htaccess' foi enviado para a raiz da 'public_html'.
